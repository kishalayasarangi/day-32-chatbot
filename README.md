# Day 32 — Simple Chatbot (Rule Based)

**120 Days of Code** | NIT Rourkela | Mechanical Engineering

## What this is
A rule-based chatbot with pattern matching, memory,
mood detection and a clean chat UI.

## How to view
Open `index.html` in your browser.

## Features
- Pattern matched responses for 50+ intents
- Remembers your name
- Mood detection
- Typing animation
- Chat history with timestamps
